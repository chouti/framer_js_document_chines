Print

打印可以使你在运行时检查变量。他和console.log的功能是类似的。只是使用print的时候输出会直接在你的原型中打印出来。注意，在 JavaScript 中你需要使用print()（不过在 CoffeeScript 中不用）。我们在接下来的文档中会忽略这点。

print "Hello"
# Output: "Hello"

你可以检查任何一种值，甚至一次多个值。

layerA = new Layer
  x: 10
  y: 20
# A single property value
print layerA.x
# Output: 10

# Multiple values
print layerA.x, print layerA.y
# Output: 10, 20
