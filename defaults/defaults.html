简介

Defaults

Framer 的 Defaults 可以让你创建图层和动画时改写他们的默认属性。打个比方，所有新建图层会得到一个淡蓝色的背景色才能让你看到它。你可以把它改成别的颜色。

# Override the default background color for layers
Framer.Defaults.Layer.backgoundColor = "red"

# Override the default corner radius for layers
Framer.Defaults.Layer.borderRadius = 10

layerA = new Layer

print layerA.backgroundColor
# Output: "red"

print layerA.borderRadius
# Output: 10

接下来这个例子是设置默认动画曲线，注意除非你为了layer.states.animationOptions重写了，否则这些也会在layer.states转换中用到。

# Override the default animation options for all Animations
Framer.Defaults.Animation =
  curve: "spring(100,10,0)"

# Override the default corner radius for layers
Framer.Defaults.Layer.borderRadius = 10

layerA = new Layer

layerA.animate
  properties:
    x: 100
# The animation will now use the "spring(100, 10, 0)" curve
