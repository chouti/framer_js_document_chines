简介

Framer JS 是可以让产品设计师（Product Designer）利用 Sketch 或者 PhotoShop 创建的产品原型图来制作可交互原型的工具。

Framer JS 利用了 CoffeeScript 和浏览器来模拟与用户的交互动作。

创造 Framer JS 的公司还发行了一款编辑器（或者说 IDE）叫 Framer Studio，卖 99 美刀。虽然贵，但是一来 Framer JS 本身是开源的，你并不必须拥有 Framer Studio 来创作，二来磨刀不误砍柴工购买了之后你会发现可以提高你的工作效率，毕竟有一些补全啊提示啊检查啊什么的功能。

这个站点我主要是想用来先对 Framer JS 的文档做一些简体中文的翻译工作。希望可以帮助到对 Framer JS 有兴趣的人。
