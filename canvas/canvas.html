Introduction

Canvas

画布对象包含了当前整个文件的像素尺寸。你改变当前对象所在窗口大小的时候画布的大小会发生改变。

Canvas.width <number>

当前整个文档的像素宽度，这是一个只读属性的值。

print Canvas.width
# Output: 640

Canvas.height <number>

当前整个文档的像素高度，这是一个只读属性的值。

print Canvas.height
# Output: 480

Canvas.size <object>

当前整个文档的像素宽高，这是一个只读属性的值。

print Canvas.size
# Output: { width:640, height: 480 }
