简介

Layers

图层是 Framer 的基本容器。图层有位置、大小、视觉属性和等级。图层可以包含图片、视频、文字或者任意的 html。

你可以用 new 这个关键词来创建图层。图层拥有一组默认属性比如蓝色以及宽度和高度。你可以在 Framer.Defaults 中改写他们。

layerA = new Layer

你可以在创建图层的时候或者创建完毕之后设置图层的所有属性。

layerA = new Layer
  x: 100
  y: 100
  width: 250
  height: 250
  opacity: 0.5
  backgroundColor: "red"

layerA.y = 900
layerA.opacity = 1

layer.id <number>

此图层的唯一编号，其他图层的编号必定与这个不同。这个属性是只读且不可更改的。

layerA = new Layer
print layerA.id
# Output: 1

layer.name <string>

图层的名字，这个属性不是默认就有的，不过你可以自己设置。从 Sketch 和 Photoshop 导入的图层会从来源图层中获得这个属性。

layerA = new Layer
layerA.name = "Button"

print layerA.name
# Output: "Button"

layer.x <number>

图层的 x 属性定义了图层相对于左上角的 x 位置。

layerA = new Layer
layerA.x = 500

layer.y <number>

图层的 y 属性定义了图层相对于左上角的 y 位置。

layerA = new Layer
layerA.y = 500

layer.z <number>

图层的 z 属性定义了它在空间中的位置，也就是常说的深度。值越大，对象越远离视图的点。

如果你需要看到深度效果记住你需要允许父图层的透视感。也要注意 z 属性不同于 layer.index （ z 轴顺序 ），定义的所有对象拥有相同的 z 值。

layerA = new Layer
layerA.z = 500

layer.width <number>

图层的像素宽度。

layerA = new Layer
layerA.width = 500

layer.height <number>

图层的像素高度。

layerA = new Layer
layerA.height = 500

layer.minX <number>

图层左边缘的位置，等同于 layer.x

layerA = new Layer
    x: 100
    y: 100
    width: 100
    height: 100

print layerA.minX
# Output: 100

layer.midX <number>

图层的水平中央。

layerA = new Layer
    x: 100
    y: 100
    width: 100
    height: 100

print layerA.midX
# Output: 150

layerA.midX = 500
print layerA.x
# Output: 450

layer.maxX <number>

图层右边缘的位置。

layerA = new Layer
    x: 100
    y: 100
    width: 100
    height: 100

print layerA.maxX
# Output: 200

layerA.maxX = 500
print layerA.x
# Output: 400

layer.minY <number>

图层上边缘的位置，等同于 layer.y

layerA = new Layer
    x: 100
    y: 100
    width: 100
    height: 100

print layerA.minY
# Output: 100

layer.midY <number>

图层的垂直中心。

layerA = new Layer
    x: 100
    y: 100
    width: 100
    height: 100

print layerA.midY
# Output: 150

layerA.midY = 500
print layerA.y
# Output: 450

layer.maxY <number>

图层底边的位置。

layerA = new Layer
    x: 100
    y: 100
    width: 100
    height: 100

print layerA.maxY
# Output: 200

layerA.maxY = 500
print layerA.y
# Output: 400

layer.point <object>

允许你设置或者捕捉图层的 x 和 y 的值。

layerA = new Layer

print layerA.point
# Output: { x: 100, y: 100 }

layerA.point =
  x: 10
  y: 200

print layerA.point
# Output: { x: 10, y: 200 }

print layerA.x
# Output: 10

layer.size <object>

允许你设置或者捕捉图层的高度和宽度。

layerA = new Layer
    width: 100
    height: 100

print layerA.size
# Output: { width: 100, height: 100}

layerA.size =
  width: 10
  height: 10

print layerA.size
# Output: { width: 10, height: 10 }

print layerA.width
# Output: 10

layer.frame <object>

允许你设置或者捕捉图层的 x、y、宽和高。

layerA = new Layer

print layerA.frame
# Output: { x: 100, y: 100, width: 100, height: 100 }

layerA.frame =
  x: 10
  y: 200
  width: 10
  height: 10

print layerA.frame
# Output: { x: 10, y: 200, width: 10, height: 10 }

print layerA.x
# Output: 10

layer.properties <object>

获取或者设置图层的所有属性。

layerA = new Layer

print layer.properties
# Output: { x: 100, y: 100, ... }

layer.center()

将当前图层置于它的父图层正中。如果没有父图层则根据屏幕居中。

layerA = new Layer
    width: 500
    height: 500

layerB = new Layer
    superLayer: layerA
    width: 100
    height: 100

layerB.center()

print layerB.x, layerB.y
# Output: 200, 200

layer.centerX(offset <number>)

将当前图层置于其父图层的水平中央。如果没有父图层则会相对于屏幕置于水平中央。Offset 的值是相对于正中的一个偏离像素值，是可选的。

layerA = new Layer
    width: 500
    height: 500

layerB = new Layer
    superLayer: layerA
    width: 100
    height: 100

layerB.centerX()
print layerB.x, layerB.y
# Output: 200, 0

layerB.centerX(20)
print layerB.x, layerB.y
# Output: 220, 0

layer.centerY(offset <number>)

将当前图层置于其父图层的垂直中央。如果没有父图层则会相对于屏幕置于垂直中央。Offset 的值是相对于正中的一个偏离像素值，是可选的。

layerA = new Layer
    width: 500
    height: 500

layerB = new Layer
    width: 100
    height: 100

layerB.centerY()
print layerB.x, layerB.y
# Output: 0, 200

layerB.centerY(20)
print layerB.x, layerB.y
# Output: 0, 220

layer.pixelAlign()

将当前图层的属性数值取整。

layerA = new Layer
    x: 100.18293
    y: 10.12873

layerA.pixelAlign()

print layerA.x, layerA.y
# Output: 100, 10

layer.screenFrame <object>

允许你设置或者捕捉当前图层相对于屏幕的位置，忽略图层的父图层继承的坐标系。或者图层在屏幕上的绝对位置。

layerA = new Layer
    x: 100

layerB = new Layer
    superLayer: layerA
    x: 100

print layerB.screenFrame
# Output: { x: 200, y: 0, width: 100, height: 100 }

layerB.screenFrame =
  x: 400
  y: 0
  width: 100
  height: 100

print layerB.x
# Output: 300

layer.contentFrame()

返回: Object 计算所有子图层相加的总大小。

layerA = new Layer
layerB = new Layer
  superLayer: layerA
  x: 0
  y: 100

layerC = new Layer
  superLayer: layerA
  x: 100
  y: 300

print layerA.contentFrame()
# Output: { x: 0, y: 0, width: 400, height: 100 }

layer.centerFrame()

返回: Object 计算在当天图层的父图层中居中，如果没有父图层，则相对于屏幕居中。

layerA = new Layer
  width: 500
  height: 500

layerB = new Layer
    superLayer: layerA
    width: 100
    height: 100

print layerB.centerFrame()
# Output: { x: 200, y: 200, width: 100, height: 100 }

layer.backgroundColor <string>

设置当前图层的背景色。颜色是在 CSS 颜色格式 定义的字符串。为了让你看得到，图层的默认背景色是淡蓝色的。

layerA = new Layer

layerA.backgroundColor = "red"
layerA.backgroundColor = "#00ff00"
layerA.backgroundColor = "rgba(134,12,64,0.4)"
layerA.backgroundColor = "transparent"

# Remove the backgroundColor
layerA.backgroundColor = ""

layer.image <string>

设定当前图层的背景图像 url 或者路径。你可以用一个本地路径或者一个完整的 url。如果图像是在线存储的可能需要花费一些时间来载入。图像会始终适合图层的大小，不会被拉伸，给一个图层设置了背景图像会移除背景色属性。你可以通过设置图像为 null 或者一个空字符串来移除一个图像。本地图片不会使用浏览器缓存。

你可能注意到当图片载入完成且准备好与Events.ImageLoaded事件一起展示。如果载入图片出错会抛出Events.ImageLoadError事件。

layerA = new Layer

layerA.image = "images/logo.png"
layerA.image = "http://framerjs.com/logo.png"

# Listen to the loading event
layerA.on Events.ImageLoaded, ->
  print "the image is loaded"

layerA.on Events.ImageLoadError, ->
  print "the image could not be loaded"

layerA.image = "images/other-logo.png"

layer.visible <boolean>

设置当前图层的可见性。

layerA = new Layer
layerA.visible = false

layer.opacity <number>

设置当前图层的透明度。透明度由一个0到1的数定义，0是全透明，1是不透明。

layerA = new Layer
layerA.opacity = 0.5

layer.clip <boolean>

设置当前图层是否应该裁剪它的子图层。裁剪是默认开启的。

注意圆角矩形的剪裁可能有缺陷，因为浏览器不能用硬件加速来渲染路径。如果你需要的话可以针对所有需要被裁剪的子图层启用layer.force2d。

layerA = new Layer
  width: 100
  height: 100

layerB = new Layer
  width: 200
  height: 200
  superLayer: layerA

layerA.clip = false

layer.ignoreEvents <boolean>

设置当前图层是否响应用户事件，比如点击或者触摸。当设置为不响应时，基于此图层的用户事件将不产生。

默认值是打开的，因为当图层不需要再去关注事件时性能会好一些。当你加入一个事件监听时 Framer 会自动将其禁止。

layerA = new Layer

layerA.on Events.click, ->
  print "Click!"

# Now it won't respond to a click
layerA.ignoreEvents = true

# Now it will
layerA.ignoreEvents = false

layer.force2d <boolean>

设置当前图层是不是只渲染 2D 属性。这会绕过 GPU 的加速渲染所以对动画来说会慢一些。当这个属性被启用后你将无法在 3D 空间内旋转。这个属性的主要目的是为了让高级遮罩功能起效，比如那些圆角的遮罩。

layerA = new Layer
  width: 400
  height: 400
  backgroundColor: "red"

layerB = new Layer
  width: 300
  height: 300
  backgroundColor: "blue"

layerB.borderRadius = layerB.width / 2

layerA.force2d = true
layerA.superLayer = layerB
