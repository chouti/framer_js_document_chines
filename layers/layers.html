简介

Layers

图层是 Framer 的基本容器。图层有位置、大小、视觉属性和等级。图层可以包含图片、视频、文字或者任意的 html。

你可以用 new 这个关键词来创建图层。图层拥有一组默认属性比如蓝色以及宽度和高度。你可以在 Framer.Defaults 中改写他们。

layerA = new Layer

你可以在创建图层的时候或者创建完毕之后设置图层的所有属性。

layerA = new Layer
  x: 100
  y: 100
  width: 250
  height: 250
  opacity: 0.5
  backgroundColor: "red"

layerA.y = 900
layerA.opacity = 1

layer.id <number>

此图层的唯一编号，其他图层的编号必定与这个不同。这个属性是只读且不可更改的。

layerA = new Layer
print layerA.id
# Output: 1

layer.name <string>

图层的名字，这个属性不是默认就有的，不过你可以自己设置。从 Sketch 和 Photoshop 导入的图层会从来源图层中获得这个属性。

layerA = new Layer
layerA.name = "Button"

print layerA.name
# Output: "Button"

layer.x <number>

图层的 x 属性定义了图层相对于左上角的 x 位置。

layerA = new Layer
layerA.x = 500

layer.y <number>

图层的 y 属性定义了图层相对于左上角的 y 位置。

layerA = new Layer
layerA.y = 500

layer.z <number>

图层的 z 属性定义了它在空间中的位置，也就是常说的深度。值越大，对象越远离视图的点。

如果你需要看到深度效果记住你需要允许父图层的透视感。也要注意 z 属性不同于 layer.index （ z 轴顺序 ），定义的所有对象拥有相同的 z 值。

layerA = new Layer
layerA.z = 500

layer.width <number>

图层的像素宽度。

layerA = new Layer
layerA.width = 500

layer.height <number>

图层的像素高度。

layerA = new Layer
layerA.height = 500

layer.minX <number>

图层左边缘的位置，等同于 layer.x

layerA = new Layer
    x: 100
    y: 100
    width: 100
    height: 100

print layerA.minX
# Output: 100

layer.midX <number>

图层的水平中央。

layerA = new Layer
    x: 100
    y: 100
    width: 100
    height: 100

print layerA.midX
# Output: 150

layerA.midX = 500
print layerA.x
# Output: 450

layer.maxX <number>

图层右边缘的位置。

layerA = new Layer
    x: 100
    y: 100
    width: 100
    height: 100

print layerA.maxX
# Output: 200

layerA.maxX = 500
print layerA.x
# Output: 400

layer.minY <number>

图层上边缘的位置，等同于 layer.y

layerA = new Layer
    x: 100
    y: 100
    width: 100
    height: 100

print layerA.minY
# Output: 100

layer.midY <number>

图层的垂直中心。

layerA = new Layer
    x: 100
    y: 100
    width: 100
    height: 100

print layerA.midY
# Output: 150

layerA.midY = 500
print layerA.y
# Output: 450

layer.maxY <number>

图层底边的位置。

layerA = new Layer
    x: 100
    y: 100
    width: 100
    height: 100

print layerA.maxY
# Output: 200

layerA.maxY = 500
print layerA.y
# Output: 400

layer.point <object>

允许你设置或者捕捉图层的 x 和 y 的值。

layerA = new Layer

print layerA.point
# Output: { x: 100, y: 100 }

layerA.point =
  x: 10
  y: 200

print layerA.point
# Output: { x: 10, y: 200 }

print layerA.x
# Output: 10

layer.size <object>

允许你设置或者捕捉图层的高度和宽度。

layerA = new Layer
    width: 100
    height: 100

print layerA.size
# Output: { width: 100, height: 100}

layerA.size =
  width: 10
  height: 10

print layerA.size
# Output: { width: 10, height: 10 }

print layerA.width
# Output: 10

layer.frame <object>

允许你设置或者捕捉图层的 x、y、宽和高。

layerA = new Layer

print layerA.frame
# Output: { x: 100, y: 100, width: 100, height: 100 }

layerA.frame =
  x: 10
  y: 200
  width: 10
  height: 10

print layerA.frame
# Output: { x: 10, y: 200, width: 10, height: 10 }

print layerA.x
# Output: 10

layer.properties <object>

获取或者设置图层的所有属性。

layerA = new Layer

print layer.properties
# Output: { x: 100, y: 100, ... }

layer.center()

将当前图层置于它的父图层正中。如果没有父图层则根据屏幕居中。

layerA = new Layer
    width: 500
    height: 500

layerB = new Layer
    superLayer: layerA
    width: 100
    height: 100

layerB.center()

print layerB.x, layerB.y
# Output: 200, 200

layer.centerX(offset <number>)

将当前图层置于其父图层的水平中央。如果没有父图层则会相对于屏幕置于水平中央。Offset 的值是相对于正中的一个偏离像素值，是可选的。

layerA = new Layer
    width: 500
    height: 500

layerB = new Layer
    superLayer: layerA
    width: 100
    height: 100

layerB.centerX()
print layerB.x, layerB.y
# Output: 200, 0

layerB.centerX(20)
print layerB.x, layerB.y
# Output: 220, 0

layer.centerY(offset <number>)

将当前图层置于其父图层的垂直中央。如果没有父图层则会相对于屏幕置于垂直中央。Offset 的值是相对于正中的一个偏离像素值，是可选的。

layerA = new Layer
    width: 500
    height: 500

layerB = new Layer
    width: 100
    height: 100

layerB.centerY()
print layerB.x, layerB.y
# Output: 0, 200

layerB.centerY(20)
print layerB.x, layerB.y
# Output: 0, 220

layer.pixelAlign()

将当前图层的属性数值取整。

layerA = new Layer
    x: 100.18293
    y: 10.12873

layerA.pixelAlign()

print layerA.x, layerA.y
# Output: 100, 10

layer.screenFrame <object>

允许你设置或者捕捉当前图层相对于屏幕的位置，忽略图层的父图层继承的坐标系。或者图层在屏幕上的绝对位置。

layerA = new Layer
    x: 100

layerB = new Layer
    superLayer: layerA
    x: 100

print layerB.screenFrame
# Output: { x: 200, y: 0, width: 100, height: 100 }

layerB.screenFrame =
  x: 400
  y: 0
  width: 100
  height: 100

print layerB.x
# Output: 300

layer.contentFrame()

返回: Object 计算所有子图层相加的总大小。

layerA = new Layer
layerB = new Layer
  superLayer: layerA
  x: 0
  y: 100

layerC = new Layer
  superLayer: layerA
  x: 100
  y: 300

print layerA.contentFrame()
# Output: { x: 0, y: 0, width: 400, height: 100 }

layer.centerFrame()

返回: Object 计算在当天图层的父图层中居中，如果没有父图层，则相对于屏幕居中。

layerA = new Layer
  width: 500
  height: 500

layerB = new Layer
    superLayer: layerA
    width: 100
    height: 100

print layerB.centerFrame()
# Output: { x: 200, y: 200, width: 100, height: 100 }

layer.backgroundColor <string>

设置当前图层的背景色。颜色是在 CSS 颜色格式 定义的字符串。为了让你看得到，图层的默认背景色是淡蓝色的。

layerA = new Layer

layerA.backgroundColor = "red"
layerA.backgroundColor = "#00ff00"
layerA.backgroundColor = "rgba(134,12,64,0.4)"
layerA.backgroundColor = "transparent"

# Remove the backgroundColor
layerA.backgroundColor = ""

layer.image <string>

设定当前图层的背景图像 url 或者路径。你可以用一个本地路径或者一个完整的 url。如果图像是在线存储的可能需要花费一些时间来载入。图像会始终适合图层的大小，不会被拉伸，给一个图层设置了背景图像会移除背景色属性。你可以通过设置图像为 null 或者一个空字符串来移除一个图像。本地图片不会使用浏览器缓存。

你可能注意到当图片载入完成且准备好与Events.ImageLoaded事件一起展示。如果载入图片出错会抛出Events.ImageLoadError事件。

layerA = new Layer

layerA.image = "images/logo.png"
layerA.image = "http://framerjs.com/logo.png"

# Listen to the loading event
layerA.on Events.ImageLoaded, ->
  print "the image is loaded"

layerA.on Events.ImageLoadError, ->
  print "the image could not be loaded"

layerA.image = "images/other-logo.png"

layer.visible <boolean>

设置当前图层的可见性。

layerA = new Layer
layerA.visible = false

layer.opacity <number>

设置当前图层的透明度。透明度由一个0到1的数定义，0是全透明，1是不透明。

layerA = new Layer
layerA.opacity = 0.5

layer.clip <boolean>

设置当前图层是否应该裁剪它的子图层。裁剪是默认开启的。

注意圆角矩形的剪裁可能有缺陷，因为浏览器不能用硬件加速来渲染路径。如果你需要的话可以针对所有需要被裁剪的子图层启用layer.force2d。

layerA = new Layer
  width: 100
  height: 100

layerB = new Layer
  width: 200
  height: 200
  superLayer: layerA

layerA.clip = false

layer.ignoreEvents <boolean>

设置当前图层是否响应用户事件，比如点击或者触摸。当设置为不响应时，基于此图层的用户事件将不产生。

默认值是打开的，因为当图层不需要再去关注事件时性能会好一些。当你加入一个事件监听时 Framer 会自动将其禁止。

layerA = new Layer

layerA.on Events.click, ->
  print "Click!"

# Now it won't respond to a click
layerA.ignoreEvents = true

# Now it will
layerA.ignoreEvents = false

layer.force2d <boolean>

设置当前图层是不是只渲染 2D 属性。这会绕过 GPU 的加速渲染所以对动画来说会慢一些。当这个属性被启用后你将无法在 3D 空间内旋转。这个属性的主要目的是为了让高级遮罩功能起效，比如那些圆角的遮罩。

layerA = new Layer
  width: 400
  height: 400
  backgroundColor: "red"

layerB = new Layer
  width: 300
  height: 300
  backgroundColor: "blue"

layerB.borderRadius = layerB.width / 2

layerA.force2d = true
layerA.superLayer = layerB

layer.originX <number>

为了放大、旋转或者倾斜变形而设置的水平原点。比如设置完这个点之后图层绕着这个点旋转。原点用介于 0 和 1 之间的数来定义，0 代表是最左侧的边，1 自然就是最右侧的边。默认值是 0.5 也就是中间。

layerA = new Layer
layerA.rotation = 45
layerA.originX = 0
layerA.originX = 1

layer.originY <number>

为了放大、旋转或者倾斜变形而设置的垂直原点。比如设置完这个点之后图层绕着这个点旋转。原点用介于 0 和 1 之间的数来定义，0 代表是最顶部的边，1 自然就是最底部的边。默认值是 0.5 也就是中间。

layerA = new Layer
layerA.rotation = 45
layerA.originY = 0
layerA.originY = 1

layer.perspective <number>

为子图层设置透视。透视给 3D 属性以深度，比如 rotationX 和 rotationY。旋转的值从 1 到无穷，1 代表了无限大的透视。设置透视为 0 你可以得到一个立方体的效果。默认的透视是关闭的。

因为 Safari 的渲染原因，透视属性可以关闭视网膜画质的渲染。我想这是以后会解决的一个缺陷。虽然在使用旋转或者动画的时候你可能不太感觉得到这个问题。

layerA = new Layer

# Set the perspective for all sub layers.
layerA.perspective = 100

layerB = new Layer
  superLayer: layerA
  rotationX: 30
  rotationY: 30

layer.rotation <number>

设置图层相对于原始形态旋转的程度。旋转是用介于 0 到 360 之间的数值来定义角度的。

layerA = new Layer
layerA.rotation = 45

layer.rotationX <number>

设置图层相对于原始形态在 x 轴方向上做了多少变形。旋转用介于 0 到 360 之间的数值来定义角度。

layerA = new Layer
layerA.rotationX = 45

layer.rotationY <number>

设置图层相对于原始形态在 y 轴方向上做了多少变形。旋转用介于 0 到 360 之间的数值来定义角度。

layerA = new Layer
layerA.rotationY = 45

layer.rotationZ <number>

设置图层相对于原始形态在 z 轴方向上做了多少变形。旋转用介于 0 到 360 之间的数值来定义角度。z 轴的旋转和 layer.rotation 是相同的。

layerA = new Layer
layerA.rotationZ = 45

layer.scale <number>

设置图层相对于原始形态的比例变化。默认的比例是 1。任何比 1 小的数字都会使得图层缩小，反之则放大。

layerA = new Layer
layerA.scale = 2
layerA.scale = 0.5

layer.scaleX <number>

设置图层相对于原始形态在横向的比例变化。默认的比例是 1。任何比 1 小的数字都会使得图层变小，而反之则放大。

layerA = new Layer
layerA.scaleX = 2
layerA.scaleX = 0.5

layer.scaleY <number>

设置图层相对于原始形态在纵向的比例变化。默认的比例是 1。任何比 1 小的数字都会使得图层变小，而反之则放大。

layerA = new Layer
layerA.scaleY = 2
layerA.scaleY = 0.5

layer.superLayer <Layer Object>

设置当前图层的父图层。如果你想让当前图层在你整个文档的根目录下存在的话，将其父图层设置为 null。

layerA = new Layer
layerB = new Layer

layerB.superLayer = layerA

print layerB.superLayer
# Output: <Object: Layer layerA>

layer.subLayers <array>

当前图层的所有子图层。

layerA = new Layer

layerB = new Layer
  superLayer: layerA

layerC = new Layer
  superLayer: layerA

print layerA.subLayers
# Output: [<Object:Layer layerB>,<Object:Layer layerC>]

layer.subLayersByName(name <string>)

返回包含了图层对象的数组 将当前图层的所有子图层按照名字过滤。

layerA = new Layer

layerB = new Layer
  name: "navigation"
  superLayer: layerA

layerC = new Layer
  name: "button"
  superLayer: layerA

print layerA.subLayersByName("button")
# Output: [<Object: Layer layerC>]

layer.addSubLayer(Layer <Layer object>)

将一个图层添加为当前图层的子图层。这个操作会修改被添加的图层的superLayer属性。

layerA = new Layer
layerB = new Layer

layerA.addSubLayer(layerB)

print layerB.superLayer
#Output: <Object:Layer layerA>

Layer.removeSubLayer(layer <Layer Object>)

将当前图层从其父图层处移除。

layerA = new Layer
layerB = new Layer
  superLayer: layerA

layerA.removeSubLayer(layerB)

print layerB.superLayer
# Output: null

layer.siblingLayers()

返回：包含了图层对象的数组 与当前图层拥有同一个父图层的图层集合。所以这些图层在层级关系中处于同一层级。

layerA = new Layer

layerB = new Layer
  superLayer: layerA

layerC = new Layer
  superLayer: layerA

print layerB.siblingLayers()
#Output: [<Object: Layer layerC>]

layer.index <index>

图层的索引顺序。同层级的图层中虽然 z 轴坐标相同，但是拥有高索引的图层会画在低索引的图层上方，低索引的在更下方。

图层索引按照插入的顺序递增。所以，如果你将一个图层作为子图层添加给一个父图层时，若最高索引的子图层为 5，则新加入的子图层会拥有索引为 6。或者说，后插入的图层始终在最上方。

layerA = new Layer
layerB = new Layer

#Draw layerB on top
layerA.index = 2
layerB.index = 1

layer.placeBefore(layer <Layer object>)

将当前图层置于另一图层的上方。这将会改变至少一个图层的layer.index属性。这个方法只对拥有同样父图层的图层起效，或者索性完全没有父图层也可以。

layerA = new Layer
layerB = new Layer

# Draw layerB on top
layerB.placeBefore(layerA)

layer.placeBehind(layer<Layer object>)

将当前图层置于另一图层的后方。这将会改变至少一个图层的layer.index属性。这个方法只对拥有同样父图层的图层起效，或者索性完全没有父图层也可以。

layerA = new Layer
layerB = new Layer

# Draw layerB on top
layerA.placeBehind(layerB)

layer.bringToFront()

将当前的图层置于所有位于同一父图层下其他图层之上。

layerA = new Layer
layerB = new Layer
layerC = new Layer

# Draw layerA on top
layerA.bringToFront()

layer.sendToBack()

将当前的图层置于所有位于同一父图层下其他图层之下。

layerA = new Layer
layerB = new Layer
layerC = new Layer

# Draw layerC last
layerC.sendToBack()

layer.html<html>

向当前图层插入任意的 html。这段 html 可以做任何事情，从简单的文本到表单内容或者画布或者 svg 内容。

如果你需要定位任意一个已经创建了的元素，记住：只有当 Framer 渲染结束（就好像 jQuery 中的 document.ready）你才能访问到它。在layer.html中可靠地索引一个 dom 元素，用layer.querySelector或者layer.querySelectorAll。

如果插入的 html 元素需要与用户交互，聪明的办法是将layer.ignoreEvents设置为false。Framer 会自动地为某些元素设置这一项。注意当你第一次设定 html 的时候，保有的图层结构会在创建时插入到一个特殊的元素中去。

layerA = new Layer

# Add simple text content
layerA.html = "Hello"

# Add inline styled text content
layerA.html = "I'm <span styled='color:red'>Kaen</span>"

#Add an input field
layerA.html = "<input type='text' value='Hello'>"

#Add a div with a canvas element and get a reference
layerA.html = "<div><canvas id='canvas'></canvas></div>"
canvasElement = layerA.querySelectorAll('#canvas')

layer.style<object>

设置或者取得图层的 css 属性。

大多数的形式属性都由 Framer 内部管理，所以当你使用时首选是用这些形式属性（比如 layer.backgroundColor 对比 layer.style["background-Color"]）。这么做的原因是从 dom 中读取数值会比较慢，为了让动画快一点我们存了一份缓存。

除了 css 标准的属性命名之外，你可以用驼峰式命名法。所以layer.style["border-color"]和layer.style.borderColor是一样的。完整的列表可以看这一份文档

layerA = new Layer

# Modify a single style property.
layerA.style["outline"] = "1px solid red"

# Modify set of style properties.
layerA.style =
  "outline": "1px solid red",
  "padding": "10px"

# Get a specific style property.
print layerA.style["outline"]
# Output: "1px solid red"
